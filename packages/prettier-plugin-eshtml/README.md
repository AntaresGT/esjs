# prettier-plugin-eshtml

Formatea tu código EsHTML con Prettier.

## Instalación

```bash
$ npm install prettier-plugin-eshtml -D
$ yarn add prettier-plugin-eshtml -D
$ pnpm add prettier-plugin-eshtml -D
```

## Uso

```ts
// .prettierrc.js
module.exports = {
	plugins: [
		// Agrega el plugin
		"prettier-plugin-eshtml",
	],
};
```
