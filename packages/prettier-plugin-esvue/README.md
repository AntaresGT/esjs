# prettier-plugin-esvue

Formatea tu código EsVue con Prettier.

## Instalación

```bash
$ npm install prettier-plugin-esvue -D
$ yarn add prettier-plugin-esvue -D
$ pnpm add prettier-plugin-esvue -D
```

## Uso

```ts
// .prettierrc.js
module.exports = {
	plugins: [
		// Agrega el plugin
		"prettier-plugin-esvue",
	],
};
```
