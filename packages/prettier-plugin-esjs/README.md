# prettier-plugin-esjs

Formatea tu código EsJS con Prettier.

## Instalación

```bash
$ npm install prettier-plugin-esjs -D
$ yarn add prettier-plugin-esjs -D
$ pnpm add prettier-plugin-esjs -D
```

## Uso

```ts
// .prettierrc.js
module.exports = {
	plugins: [
		// Agrega el plugin
		"prettier-plugin-esjs",
	],
};
```
